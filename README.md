# Meteor Scatter Radio Source Materials and References

Links and references for the Meteor Scatter Radio project! <3
The goal of this project is to use machine learning techniques to identify and classify meteor scatter radio signals from radio observations. 

### *_From Radio Meteor Zoo:_*

The Geminids are a prolific meteor shower caused by the object 3200 Phaethon, which is thought to be an asteroid of the same type as 2 Pallas. This would make the Geminids, together with the Quadrantids, the only major meteor showers not originating from a comet.
The meteors from this shower are much slower (35 km/s) than most other meteor showers, can be seen in December and peak around December 7-17. Letâ€™s see if we can confirm this with our radio observations!
